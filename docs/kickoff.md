# Kickoff — Projeto de Automação de Dados em Plano de Saúde

## 1. Contexto do Negócio

A empresa contratante é uma administradora de benefícios de saúde corporativos, responsável pela gestão de planos de saúde para mais de 800 empresas clientes.

Atualmente, a operação envolve o controle de milhares de titulares e dependentes, além da gestão de mais de 100 fornecedores e prestadores de serviço.

Grande parte dos processos operacionais é realizada de forma manual, utilizando planilhas Excel descentralizadas, o que gera riscos operacionais, retrabalho e dificuldade na consolidação de informações.

## 2. Problemas Identificados

Durante o diagnóstico inicial, foram identificados os seguintes problemas:

- Alto volume de planilhas manuais, sem padronização
- Duplicidade de registros de beneficiários
- Dificuldade no controle de cancelamentos e movimentações
- Falta de rastreabilidade das alterações
- Erros recorrentes em cobranças e repasses
- Dependência excessiva de processos manuais

## 3. Objetivo do Projeto

O objetivo deste projeto é automatizar o fluxo de dados operacionais da empresa, centralizando as informações em uma base estruturada e confiável.

A automação visa reduzir erros manuais, melhorar a rastreabilidade das informações, acelerar os processos operacionais e fornecer dados confiáveis para tomada de decisão através de dashboards.

## 4. Escopo Inicial

O escopo inicial do projeto inclui:

- Organização e padronização das bases de dados
- Tratamento de dados brutos provenientes de planilhas Excel
- Modelagem de dados utilizando SQL
- Criação de rotinas de automação em Python
- Desenvolvimento de dashboards no Power BI
